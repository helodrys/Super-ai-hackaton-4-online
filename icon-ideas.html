Wandr Marker System

Pin style showcase for your travel map

Below are multiple marker directions you can use in your current MapLibre + HTML-marker setup. Each one is shown visually so you can pick based on mood, clarity, and brand feel.

Best usability: Category Icon
Best premium feel: Luxury Numbered
Best futuristic UI: Glass Badge
Best info density: Chip Marker
1

1. Luxury Numbered Pin

Premium

Elegant teardrop marker with route order clearly visible. Very good when your itinerary has stop 1, 2, 3, 4.

Clear route order Easy to scan Strong brand feel
🛕

2. Category Icon Pin

Most useful

Best for tourists because they instantly understand what a place is: temple, food, market, park, museum, nightlife.

Best UX Tourism-friendly Fast recognition

Modern look Clean UI Good for dashboard modes

My recommendation for Wandr:
Use Category Icon Pin as the main map marker system, and combine it with a small route order badge when needed. Then use Pulse Live Pin only for current location, and Chip Marker only when a place is selected or hovered. That gives you a clean map by default, but still feels informative and premium.
