Implementation direction for Codex

  • Replace the current bottom chips with a component called ItineraryTray or plain DOM function renderItineraryTray().
  • Each stop should include: number, name, type, reason, tip, distanceFromPrevious, and durationFromPrevious.
  • Use OSRM routes[0].legs to calculate per-stop distance and time.
  • On click, update selectedStopIndex, highlight the marker, fly to the place, and update the selected detail card.
  • Use desktop tray above 768px, and expandable drawer below 768px.
