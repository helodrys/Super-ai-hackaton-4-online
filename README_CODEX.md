# SafeFlow Thailand — Codex Build Brief

You are building a premium frontend mockup for **SafeFlow Thailand**, an AI tourism web app for Thailand.

This is a **mockup-first prototype**. Use mock data. Do not build production backend yet.

## Product in one sentence

SafeFlow Thailand helps tourists decide whether to go, delay, or reroute by checking weather, PM2.5, taxi scam risk, tourist friction, and personal travel preferences — then creates a better travel plan.

## Core pages

1. **Dashboard**
   - Weather / feels-like temperature
   - PM2.5
   - travel comfort score
   - location overview
   - entry buttons to Taxi Scam Checker and Trip Planner
   - beautiful mini map / travel condition overview

2. **Taxi Scam Checker**
   - pickup location
   - destination
   - offered price
   - suspicious behavior checkboxes
   - risk score
   - fair price range
   - action advice
   - Thai phrase helper

3. **Trip Planner**
   - traveler profile
   - accessibility / disability / child / pet / elderly options
   - allergies / halal / vegan / vegetarian
   - desired travel vibe
   - food preferences
   - budget
   - time
   - avoid preferences
   - custom prompt

4. **Itinerary Result**
   - large map overview
   - route pins
   - stop-by-stop timeline
   - budget/time/comfort/accessibility/local impact scores
   - AI explanation panel
   - alternative route versions

## Build requirements

Use:
- React or Next.js
- Tailwind CSS
- Framer Motion
- shadcn/ui or custom polished components
- mock JSON data
- responsive layout
- rich animation and microinteractions

## Quality bar

This must not look like a generic SaaS admin dashboard.

It should feel:
- premium
- colorful
- travel-inspired
- personal
- polished
- emotionally appealing
- commercially viable
- slightly game-like but still elegant

The user should want to click around.
